# Gestion Restaurant — Module Odoo 19

Module personnalisé Odoo 19 pour la gestion d'un restaurant.

## Auteur

Moussa Diakite — Étudiant en Génie Informatique, Université Mundiapolis, Casablanca

## Fonctionnalités

- Gestion des Tables (Liste, Formulaire, Kanban)
- Gestion des Plats avec lien Inventaire
- Gestion des Commandes avec lignes de commande
- Workflow : En attente > En préparation > Servie > Payée
- Vues Graph et Pivot
- Rapport PDF QWeb

## Installation

1. Copier le dossier dans le répertoire addons
2. Ajouter le chemin dans odoo.conf
3. Redémarrer Odoo et installer le module

## Modèles

- gestion.table : Tables du restaurant
- gestion.plat : Plats du menu
- gestion.ligne.commande : Lignes de commande
- gestion.commande : Commandes
